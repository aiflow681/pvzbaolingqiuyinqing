【游戏名称】
植物大战外星人-保龄球（重构版）

【游戏介绍】
这是一款基于 Phaser 引擎开发的星际主题保龄球塔防小游戏。本作在《植物大战僵尸-保龄球》的基础上，进行了彻底的代码重构与美术风格升级。玩家需要通过精准的操作，投掷星际保龄球，消灭源源不断入侵的外星僵尸，保卫星际基地的安全。

【玩法说明】
1. 目标：控制轨道上的保龄球发射器，通过投掷保龄球撞击入侵的僵尸。
2. 操作：
   - 桌面端：点击屏幕或使用两侧的导航按钮（◀ 和 ▶）调整发射位置。
   - 移动端：轻触屏幕两侧或控制按钮进行位移。
3. 关卡：随着击败的僵尸增多，外星人的入侵速度和密度会逐渐提升，极具挑战。

【技术介绍】
- 引擎切换：采用 Phaser 现代游戏引擎，支持更流畅的渲染。
- 架构优化：实现了资源懒加载、状态机管理、以及响应式的画布适配系统。
- 视觉重构：引入了深海/星际渐变渲染技术，提升了旧版简单的 Canvas 视觉体验。
- 性能优化：相比原版，大幅降低了内存占用并提升了移动端的帧率表现。
